# Template Project (Wii Homebrew)

A minimal **Nintendo Wii homebrew** game built with **devkitPro**, packaged as a **WBFS** game for USB loaders like **USB Loader GX**.

This project demonstrates:

- Writing Wii homebrew in C
- Compiling `.dol` using devkitPPC
- Packaging the game as a WBFS
- Running on a **real Wii console**

---

## Features

- Dark green background
- Red player square
- Move with **Wii Remote D-Pad**
- **Home button** exits the game
- ~30 FPS gameplay

---

## Tools Used

- [**devkitPro**](https://devkitpro.org) – Wii/GC homebrew development toolkit  
- [**libogc**](https://github.com/devkitPro/libogc) – Wii homebrew library  
- [**Wiimms ISO Tools (wit)**](https://wit.wiimm.de) – extract/build WBFS/ISO images  
- [**USB Loader GX**](https://github.com/USBLoaderGX/usbloadergx) – Wii USB loader for testing  

Optional banner editing:  

- [**CustomizeMii**](https://sourceforge.net/projects/customizemii/) – Wii banner editor  
- [**BNS Tools / Wii Banner Player**](https://github.com/DirkClemens/BNS-Tools) – edit icons, animations, and sound
